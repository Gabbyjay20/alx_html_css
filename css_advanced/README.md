# CSS Advanced

This project is part of the ALX HTML/CSS curriculum, focusing on advanced CSS techniques.

## Project Overview

Smile School is a fictional website for learning how to smile, featuring tutorials, testimonials, and membership options.

## Structure

- `index.html`: Main HTML structure for the website
- `images/`: Directory containing images used in the website

## Technologies Used

- HTML5
- CSS (advanced styling to be implemented)
- Font Awesome for icons

## Features

- Responsive navigation
- Hero section with call-to-action
- Testimonials from smile experts
- Popular tutorials with ratings
- Free membership section
- FAQ section
- Footer with social links
